# ProSeed - Transportation Management System

A comprehensive transportation management system built with Next.js, TypeScript, and Firebase. Features real-time fleet tracking, driver management, customer booking, and logistics optimization.

## Features

- Real-time vehicle tracking and fleet management
- Driver management and scheduling
- Customer booking and logistics
- AI-powered route optimization
- Multi-role authentication system
- Responsive dashboard with analytics

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Firebase, Firestore
- **UI Components**: shadcn/ui
- **Maps**: React Leaflet
- **State Management**: Zustand

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up Firebase configuration
4. Run the development server: `npm run dev`

## Project Structure

- `/src/app` - Next.js app router pages
- `/src/components` - Reusable UI components
- `/src/stores` - State management stores
- `/src/lib` - Utility functions and types
- `/src/firebase` - Firebase configuration and utilities
